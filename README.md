# AI FAQ Consultant - AI-сервис поддержки клиентов на базе FAQ (RAG)

AI FAQ Consultant - сервис для автоматизации клиентской поддержки.
Отвечает на вопросы пользователей на основе базы знаний (FAQ) с использованием Retrieval-Augmented Generation (RAG).

Подходит для e-commerce, логистики, SaaS и любых сервисов с повторяющимися вопросами пользователей.

---

## Problem

Поддержка клиентов сталкивается с типовыми проблемами:

* большое количество повторяющихся вопросов
* высокая нагрузка на операторов
* долгие ответы пользователям
* сложность масштабирования поддержки

FAQ-страницы существуют, но пользователи редко ими пользуются.

---

## Solution

AI-сервис автоматически отвечает на вопросы пользователей, используя FAQ как источник знаний:

* извлекает релевантную информацию из базы знаний
* формирует ответ через LLM
* отвечает в удобном интерфейсе (Telegram)
* логирует диалоги для анализа

Позволяет снизить нагрузку на поддержку и ускорить ответы.

---

## How it works

1. FAQ сайта парсится и сохраняется в структурированном виде
2. Данные индексируются в векторной базе (FAISS)
3. Пользователь задаёт вопрос
4. Система:

   * находит релевантные фрагменты
   * передаёт их в LLM
   * генерирует ответ строго на основе базы знаний

---

## Architecture

User → Telegram Bot → RAG Backend → FAISS (vector search) → LLM
↓
SQLite (логи)

## Как работает бот

* `/start` — запуск и меню
* `/help` — инструкция
* кнопки:

  * «Тарифы»
  * «Вызов курьера»
  * «Часы работы»

Любой вопрос:

1. отправляется в RAG-систему
2. ищутся релевантные фрагменты FAQ
3. генерируется ответ
4. возвращается пользователю

## Логирование

Все диалоги сохраняются в SQLite:

* user_id
* вопрос
* ответ
* релевантный FAQ
* время запроса

Это позволяет:

* анализировать запросы пользователей
* находить слабые места FAQ
* улучшать качество ответов
